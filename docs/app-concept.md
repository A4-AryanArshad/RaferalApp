# App Concept

## Problem Statement

**For Travelers:**
- No easy way to recommend Airbnb listings to friends
- No financial incentive for word-of-mouth recommendations
- Missed opportunity to earn rewards from their influence

**For Hosts:**
- Lack of structured word-of-mouth marketing channel
- Limited organic referral system
- Need for cost-effective customer acquisition

## Solution

A mobile application that connects travelers with hosts through a structured referral system:

1. **Travelers** can browse Airbnb listings and generate unique referral links
2. **Travelers** share these links with friends via WhatsApp, email, SMS, social media
3. **When friends book** through the referral link, the referrer earns rewards:
   - Financial compensation per validated booking
   - Free nights after reaching milestone (e.g., 5 sponsored bookings)
   - Premium Ambassador status with exclusive bonuses

## Core Features

### 1. Listing Discovery
- Browse/search Airbnb listings
- View listing details, photos, reviews
- Filter by location, price, amenities
- Save favorite listings

### 2. Referral Generation
- Generate unique referral link for any listing
- Track referral link performance
- Share via multiple channels (WhatsApp, Email, SMS, Social Media)

### 3. Rewards System
- Real-time tracking of:
  - Number of people referred
  - Rewards earned (monetary)
  - Free nights unlocked
  - Progress toward milestones
- Reward redemption options
- Transaction history

### 4. User Profile
- Account management
- Referral statistics
- Payment information
- Ambassador tier status

## Target Users

**Primary:** Airbnb travelers (18-45 years old, tech-savvy, social media active)
**Secondary:** Airbnb hosts (seeking cost-effective marketing channels)

## Value Proposition

**For Travelers:**
- Earn money by recommending places they love
- Get free nights for future stays
- Exclusive ambassador benefits

**For Hosts:**
- Access to word-of-mouth marketing channel
- Pay-per-performance model (only pay when booking occurs)
- Higher quality referrals from trusted sources

## Monetization Model

- Commission-based: Hosts pay a percentage of booking value when referral leads to confirmed booking
- Platform takes small service fee
- Remaining amount distributed to referrer as reward
